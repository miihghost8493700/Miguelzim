Tipos de Sistemas Operacionais

💻 Desktop

Ex: Microsoft Windows

Usado em computadores pessoais como PCs e notebooks.

Foco: usuário final.

📱 Mobile

Ex: Android

Usado em smartphones e tablets.

Foco: aplicativos e mobilidade.

🌐 Servidores

Computadores que fornecem serviços para outros.

Hospedam sites, como os feitos no GitHub Pages.

Foco: desempenho, segurança e estabilidade.

🤖 Embarcados

Presentes dentro de dispositivos eletrônicos.

Ex: TV, carro, micro-ondas.

Foco: executar funções específicas.
